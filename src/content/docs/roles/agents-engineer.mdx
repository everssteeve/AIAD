> **Essence** : Construire et optimiser l'écosystème d'agents qui démultiplie les capacités de l'équipe.

## Pourquoi cette responsabilité existe

Un écosystème d'agents mal configuré produit du code générique et incohérent. Bien calibré, il produit du code adapté au contexte, aux standards et aux conventions de l'équipe.

## Ce que fait l'AE

| Action | Fréquence |
|--------|-----------|
| Sélectionner les agents adaptés aux besoins | Par besoin |
| Configurer et calibrer les agents | Continu |
| Définir la gouvernance de l'écosystème | Mensuel |
| Former l'équipe aux bonnes pratiques | Hebdomadaire |
| Monitorer les performances des agents | Continu |
| Expérimenter de nouveaux agents et configurations | Hebdomadaire |

## L'écosystème à 4 niveaux

L'AE structure l'écosystème en quatre couches complémentaires :

| Niveau | Rôle | Exemples |
|--------|------|----------|
| **Gouvernance** | Assurer la cohérence globale | Règles de code, conventions, standards |
| **Qualité** | Vérifier automatiquement | Linters, analyseurs statiques, tests automatisés |
| **Productivité** | Accélérer le travail quotidien | Autocomplétion, génération de boilerplate, refactoring |
| **Agent Principal** | Générer du code métier | Agent de codage principal orchestré par le PE |

## Principe de sélection

- **Commencer minimal** : ne pas ajouter d'agent sans besoin identifié
- **Ajouter par douleur** : un nouvel agent résout un problème récurrent mesurable
- **Retirer par obsolescence** : supprimer les agents qui n'apportent plus de valeur
- **Optimiser par mesure** : chaque agent doit prouver son utilité par des métriques

## Indicateurs de performance

| Indicateur | Cible |
|------------|-------|
| Taux d'adoption des agents par les PE | Plus de 90% |
| Faux positifs des agents de qualité | Moins de 20% |
| Satisfaction des PE (sur 10) | Plus de 8/10 |

## Anti-pattern

**L'AE "collectionneur"** qui accumule les agents "au cas où" sans jamais mesurer leur impact réel. Chaque agent ajouté doit résoudre un problème concret et mesurable.
